1.  foreach and map Difference

    -> forEach → returns nothing (undefined)

       map → returns a new array,not modify old array 
    
    -> map is used to render lists, forEach is not used for rendering

    -> forEach → used to do something with each item

       map → used to transform items and get a new array

    -> Conditional Rendering Inside map
     
       -- {numbers.map(num =>
             num % 2 === 0 ? <p key={num}>{num}</p> : null
          )}


    ->   {
            cars.map((c)=>{
                return <h1>{c}</h1>   //run error free
            })

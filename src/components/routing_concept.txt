2.  routing 

    -> When the URL changes, show a different page (component).

       - /home      → Home page
         /about     → About page

    dynamic routing

    -> Part of the URL can change, but the page stays the same
       
       - /users/1
         /users/2
         /users/99
       - 1, 2, 99 are dynamic
       - The component is stil same

       - <Route path="/users/:id" element={<User />} />
       - :id → dynamic value
       - id is called a route parameter
